Introduction

The MarketSync™ Trading Platform provides the user with an advaced tool to trade any market. State of the art charts and indicators allow to keep track of market fluctuations and trends.

Interface

The main interface of the platform is divided in 3 sections: Charts, the Portfolio Monitor and the Order Section.

Charts

The chart section is by default on the left part of the screen but can be moved to any other part depending ont he user preferences or even taken out if desired.

Portfolio Monitor

The portfolio monitor is located on the top right part of the screen and provides the user with the current status and information about his portfolio, the assets that he owns and their current performance.

Orders

In the orders section, located by default in the bottom of the screen, the user can ask for quotes and place orders for any of the markets available.

Trade

Order Entry

Use the Order Entry panel to create and then Submit or Save an order. It's easy to change the asset by selecting from the drop down list of previously used instruments, or typing in a new one manually. To set up an option order, click the Option Chain button to easily pick an option contract.

Custom Algorithms

Our platform allows the user to code in their own buy and sell algorithms using a very intuitive and easy to use coding language designed exclusively for trading.

Example 1: Declaring an asset variable

#import assets.file; Asset a = $NFLX; Asset b = $USDMXN;

Example 2: Buying and selling an asset

import assets.code; import operations.code; Asset a = $NFLX; var qty = 1; operation.buy(qty, a, "market");

Example 3: Building a panic stop order

import assets.code; import operations.code; if(indexes.SP500 .dailyChange .isMoreThan(-5%){ operation.sellPortfolio(all, market); }

Example 4: Building a virtual ETF

import assets.code; import etf.code; Asset f = $FB; Asset a = $AMZN; Asset n = $NFLX; Asset g = $GOOGL; Etf $fang = etf.builder(f+a+n+g);

Example 5: Setting a tradebot

import assets.code; import bots.code; Asset a = $NFLX; Strategy myStrat = new Strategy(RSIstrat) myBot = new bot("autotrade", myStrat, daily); myBot.start();
